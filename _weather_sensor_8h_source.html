76 // Sensor Types
77 // 0 - Weather Station 5-in-1; PN 7002510..12/7902510..12
78 // 1 - Weather Station 6-in-1; PN 7002585
79 // - Professional Wind Gauge 6-in-1; PN 7002531
80 // 2 - Thermo-/Hygro-Sensor 6-in-1; PN 7009999
81 // 3 - Lightning Sensor PN 7009976
82 // 4 - Soil Moisture Sensor 6-in-1; PN 7009972
83 // 9 - Professional Rain Gauge (5-in-1 decoder)
84 // 11 - Weather Sensor 7-in-1 7-in-1; PN 7003300
85 // ? - Air Quality Sensor
86 // ? - Water Leakage Sensor
